# DreamHouse

Простий «розумний дім» у браузері: кімнати, пристрої, перемикачі, модалка з налаштуваннями

## Що вміє

- Чотири кімнати (bedroom, kitchen, living room, bathroom), у кожній свої картки пристроїв.
- Зверху показує температуру, вати й скільки пристроїв увімкнено (оновлюється кожні кілька секунд).
- Можна додати пристрій: тип, кімната, назва (форма ховається/показується кнопкою в шапці).
- Для світла, TV, жалюзі й камери є окреме вікно налаштувань.
- TV: канали, гучність, mute, вхід, пошук, додавання каналу.
- Стан усіх пристроїв пишеться в `localStorage` (ключ `dreamhouse_devices_v1`), щоб після перезавантаження нічого не губилося.


## Моделі

`Light` — яскравість, колір, режими normal/night/party.  
`TV` — канали, гучність, вхід HDMI/USB/антена тощо.  
`Blinds` — відсоток відкриття, нахил.  
`Camera` — режим, роздільність, нічний режим, симуляція руху.
